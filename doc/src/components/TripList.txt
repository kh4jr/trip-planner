#### [SECTION 2] Architectural Review & Main Logic  
- **Rola komponentu**: Komponent prezentacyjny służący do wyświetlania listy dostępnych podróży. Umożliwia użytkownikowi wybór (kliknięcie) wyjazdu, co aktualizuje widok szczegółów.
- **Typ komponentu**: Komponent funkcyjny (stateless component). Nie posiada własnego stanu wewnętrznego, jest w pełni sterowany właściwościami (props) przekazywanymi z komponentu nadrzędnego.
- **Definicja Props**:
  - `trips`: Tablica obiektów typu `FullTrip` do wyświetlenia.
  - `activeTripId`: Identyfikator wybranej aktualnie podróży (w celu wizualnego zaznaczenia elementu na liście).
  - `onSelectTrip`: Callback wywoływany w momencie kliknięcia na pozycję wyjazdu z listy.
- **Główna Logika Biznesowa**:
  - Warunkowe aplikowanie klas Tailwind CSS w zależności od stanu aktywności elementu (`activeTripId === trip.id`), co poprawia czytelność interfejsu (UX).

#### [SECTION 3] Code Review & Explanatory Comments  
```typescript
import { FullTrip } from '@/types/fullTrip';

// Definicja interfejsu określającego parametry wejściowe komponentu
interface TripListProps {
  trips: FullTrip[];
  activeTripId: number | null;
  onSelectTrip: (id: number) => void;
}

export default function TripList({
  trips,
  activeTripId,
  onSelectTrip,
}: TripListProps) {
  return (
    <div className="flex flex-col gap-2">
      {/* Iteracja po liście podróży i generowanie kafelka dla każdej z nich */}
      {trips.map(trip => (
        <div
          key={trip.id}
          // Zdarzenie kliknięcia uruchamiające callback przekazujący ID wybranego elementu
          onClick={() => onSelectTrip(trip.id)}
          // Dynamiczne przypisywanie klas stylów w zależności od statusu aktywności
          className={`p-3 rounded-xl cursor-pointer transition ${
            activeTripId === trip.id
              ? 'bg-blue-100 text-blue-700 font-bold' // Styl dla wyjazdu aktywnego
              : 'text-gray-600 hover:bg-gray-100'     // Styl dla pozostałych wyjazdów
          }`}
        >
          {trip.name}
        </div>
      ))}
    </div>
  );
}
```
